# src 目录内容摘要

1. **ai.ts**: 包含核心 AI 逻辑，用于使用预定义模型和工具生成文本响应。其中包括一个函数 `generateTextAI`，用于处理消息并生成响应。
2. **core.ts**: 定义了 Coder Demo 的主要执行逻辑。它初始化系统并通过循环处理用户提示。
3. **index.ts**: 应用程序的入口点，简单地调用 `core.ts` 中的 `run` 函数。
4. **loop.ts**: 实现了一个循环机制，用于处理与 AI 的迭代交互，处理消息并管理工具结果，直到满足条件为止。
5. **config**: AI 模型和设置的配置文件。
6. **prompt**: 包含提示生成逻辑。
7. **tools**: 定义 AI 可以使用的内置工具。
8. **typings**: 项目的类型定义。
9. **.env**: 环境变量配置。

本摘要概述了 `src` 目录中每个文件的功能和用途。
